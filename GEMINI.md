# Khrona Project Instructions

## General Workflows
Refer to [CONTRIBUTING.md](./CONTRIBUTING.md) for the following project standards and workflows:
- **Release Workflow**: Step-by-step process for versioning and tagging.

## Automated Benchmarking
Ensure that any changes to the scheduler loop or storage implementations are verified against the performance standards using the `@Disabled` tests referenced in `CONTRIBUTING.md`.
